How the GK Nifty Heads Tracker Works

01
GitOps Serverless Pipeline
GitHub Actions runs nightly for free. It fetches only new mints & burns from the WAX AtomicAssets API, recalculates rarity, and commits fresh JSON files back to this repo.
02
Pre-computed Everything
No database queries at runtime. All rarity scores, holder rankings, and trait frequencies are calculated once per day and stored as tiny static JSON files served instantly via jsDelivr CDN.
03
Rarity Formula
Standard statistical rarity: for every trait an asset has, we add total_active / frequency_of_trait. Higher score = rarer. Simple, transparent, and effective.
This exact system (or an improved version of it) can power your official portal. The code is 100% open and ready to extend.
