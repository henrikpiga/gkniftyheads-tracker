How Weighted Rarity + Surviving Ranks Work

Every night the GitHub Action re-calculates only from surviving (non-burned) assets.
Surviving Mint Rank: Assets are re-ranked 1→N within their template based on their original template_mint number among only the ones still alive.
Weighted Rarity Score = Base statistical rarity + bonuses for:
  • Smaller template max supply (template weight)
  • Very low original mint numbers (#1–#10 get big bonus)
  • "Legendary", "1/1", "Mythic" rarity name traits
  • Special variation traits
Live Trait Exposure: What % of the current surviving supply still carries each trait value. Updates automatically after every burn/mint.
All of this is pre-computed into static JSON files → instant dashboard, zero database cost.
